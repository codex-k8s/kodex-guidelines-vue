# kodex guidelines vue

Публичный репозиторий инженерных правил для Vue 3 и TypeScript frontend.

## Назначение

Этот репозиторий содержит правила проектирования и разработки frontend:

- архитектура приложения;
- работа с API, состоянием и маршрутизацией;
- обработка ошибок;
- правила кода и комментариев;
- правила выноса библиотек.

## Использование

До внедрения штатного импорта руководящих пакетов основной репозиторий `github.com/codex-k8s/kodex` хранит локальную копию этих файлов в `docs/design-guidelines/vue/**`.

Изменения в этом репозитории и локальной копии в `kodex` должны синхронизироваться в одном рабочем срезе.

## Связанные репозитории

- `github.com/codex-k8s/kodex` — основная платформа.
- `github.com/codex-k8s/kodex-guidelines-common` — общие инженерные правила.
- `github.com/codex-k8s/kodex-guidelines-go` — правила для Go backend.
