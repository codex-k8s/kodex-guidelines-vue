# Vue Frontend Guidelines

Публичный репозиторий инженерных правил для frontend-стека Vue 3 + TypeScript + Pinia + Axios + Vite.

## Назначение

Этот репозиторий содержит правила проектирования и разработки frontend:

- архитектура приложения;
- работа с API, состоянием и маршрутизацией;
- обработка ошибок;
- правила кода и комментариев;
- правила выноса библиотек.

## Использование

Репозиторий можно использовать:

- как внешний пакет frontend-правил;
- как базовый набор документов для локальной проектной копии, например в `docs/design-guidelines/vue/**` или аналогичном каталоге проекта;
- как основу для проектного overlay, где конкретный проект фиксирует свои auth/SSO, routing, UX и release conventions.
